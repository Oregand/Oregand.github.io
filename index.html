College Projects
CarDealz

CarDealz

Description

Cardealz was my 4th Year project, the equivalent of a thesis for my BSc. The idea behind the program, was to develop an application that used Machine Learning to, analysis current online car sale's across multiple sites, store the data in a web server, and finally, analysis the gathered data to evaluate which of the online ads are the best value.

The project started with a humble beginning but quickly escalated into something much more complex as time went on. Broken into three parts:

  • Data Scraping - Scrapy(Python)
  • Data Storage/Analytics(MYSQL, Python)
  • End User Display(Android/IOS)
I kept a small website and blog dedicated to the project which you can find here.
