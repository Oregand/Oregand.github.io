Project Spec

Performance/Scalability Testing for three open source PaaS's

For our project we have choosen the three following PaaS's to use:

  • Containership
  • Heroku
  • Tsuru

Our goal is to deploy a simple microservice to each of these systems inside a docker container, then perform a series of tests for performance/scalability on each.

Feature scope between the three is a secondary metric for our project.

  • Containership

  • Heroku

  • Tsuru

  • Cloud Compuitng

  • PaaS

  • Docker
